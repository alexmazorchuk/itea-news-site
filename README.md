News Portal
-----------

This is demo projects on Symfony 4 framework.

Installation
------------

1. Clone repository

    ```bash
    $ git clone https://github.com/alexmazorchuk/itea-news-site news_portal
    $ cd news_portal
    ```
    
2. Install dependencies

    ```bash
    $ composer install
    ```
    
3. Run server

    Linux
    ```
    $ symfony serve
    ```
    Windows
    ```
    $ "c:\Program Files\Symfony\symfony.exe" serve
    ```
